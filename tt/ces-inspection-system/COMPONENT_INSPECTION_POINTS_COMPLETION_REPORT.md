## ğŸ“ How to Use the Feature

### Access the Feature
1. Open the application
2. Go to "Asset Inspection Points" section
3. Click "Component Inspection Points" tab

### View Points
- Table shows all 4 sample component inspection points
- Each point displays: ID, Template, Category, Sequence, Description, Method, Mandatory status, Threshold

### Filter Points
1. Type a component category (e.g., "Electrical")
2. Table automatically filters to matching points
3. Click "Clear" to reset filter

### Add a Point
1. Scroll to "Add Component Inspection Point" form
2. Fill all required fields
3. Click "Add Component Point"
4. New point appears in table

### Delete a Point
1. Click "Delete" button on any point
2. Confirm in dialog
3. Point is removed
